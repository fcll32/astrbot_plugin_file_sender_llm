# astrbot_plugin_file_sender_llm

## 插件作者：fcll（本插件由ai辅助完成）

## 插件简介

`astrbot_plugin_file_sender_llm` 是一个面向 AstrBot 的  文件发送插件，用户可通过自然语言触发，无需记忆命令。

核心特性：

- 自然语言触发文件发送
- 平台消息下发后自动进行意图识别
- 可配置白名单根目录，限制可发送文件范围
- 支持文件名模糊匹配与候选确认
- 强调安全边界与合规设计（无危险执行、无越权访问）

---

## 功能说明

1. **LLM 工具化能力暴露**
   - 文件发送能力统一通过 `llm_tool` 暴露。
   - 无需新增命令体系，减少使用门槛。

2. **自动意图识别**
   - 当用户输入“发一下周报”“把说明文档传我”等自然语言时，模型可识别为发送文件任务。

3. **白名单目录限制**
   - 仅允许在配置目录内检索文件。
   - 防止访问非业务目录或敏感目录。

4. **模糊匹配与歧义处理**
   - 支持不完整文件名检索。
   - 单一高置信结果可直接发送；多候选时提示用户补充信息。

5. **结果反馈机制**
   - 对成功、未命中、歧义、权限拒绝等状态给出明确提示。
